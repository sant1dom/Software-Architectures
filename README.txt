Setup:

1) Clone the project from GitHub
2) Create a new Python Project in the Project Dir
3) Add the module: requests, fastapi, pydantic
4) Open the shell and go to the Project Dir
5) type "docker compose up"
6) Go on "http://localhost:8084/" for the Web Dashboard
7) Go on "http://localhost:8083/" for the API Gateway
8) Go on "http://localhost:8081/" for the Micro Service "AuthReg"
9) Go on "http://localhost:8082/" for the Micro Service "Bills"
9) Go on "http://localhost:8085/" for the Micro Service "Prediction"
10) Go on "http://localhost:8880/" for Node-RED
11) Go on "http://localhost:8080/" for Apache Kafka
12) Go on "http://localhost:8086/" for Influx DB (Credentials: admin/adminpass)
13) Go on "http://localhost:9093/" for Broker
14) Go on "http://localhost:3000/" for Grafana
